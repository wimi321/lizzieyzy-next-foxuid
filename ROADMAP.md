# Roadmap

`LizzieYzy Next` 的目标不是做一个“临时修一下就放着”的分支，而是把原来很多人还在用的 `lizzieyzy` 主链路继续维护下去。

The goal of `LizzieYzy Next` is not a one-off patch. It is a maintained continuation focused on the workflow real users still depend on.

## Current Priorities

现在最优先的事情：

- 继续保持野狐棋谱抓取可用，而且让普通用户直接输入 **野狐昵称** 就能完成
- 继续把 KataGo 整合包做成“下载、安装、打开就能分析”
- 持续改进 Windows 安装器、macOS `.dmg`、Linux 整合包的可理解性和稳定性
- 继续完善官方权重下载体验、首次启动自动设置、默认配置自动化
- 让 README、发布页、安装文档、真实安装包命名始终保持一致

## What We Want This Project To Feel Like

我们希望这个项目给用户的感受是：

- 打开 GitHub 首页就知道该下哪个包
- 第一次安装不需要先研究很多背景知识
- 遇到问题时知道去哪里提问、去哪里反馈
- 发布包不是“能跑就行”，而是尽量像一个认真维护的桌面项目

## Near-Term Work

接下来一段时间会持续推进的方向：

- 更多真实机器安装反馈，尤其是 Windows 和 Intel Mac
- 更清楚的发布页和 release notes
- 更稳定的野狐抓谱提示、错误反馈和结果展示
- 更完整的官方权重选择、下载、停止、恢复体验
- 更统一的中英文和多语言文案
- 更顺手的默认设置和首启体验

## Good Help Right Now

如果你想帮忙，这几类反馈最有价值：

- Windows 64 位安装器和便携包的真实安装反馈
- Intel Mac 和 Linux 桌面环境的实际运行反馈
- 野狐昵称抓谱的可用性反馈
- 官方权重下载体验反馈
- README、安装文档、发布页文案优化
- 打包、自动化、校验脚本相关改进

## What This Project Is Not Trying To Do First

当前阶段不会优先追求这些方向：

- 支持所有历史系统和所有第三方改版配置
- 为每一种自定义引擎组合做专门适配
- 只面向会手工改路径、改参数、改脚本的少数用户

项目当前的第一优先级，仍然是把普通用户最常走的那条链路持续维护好。
