# Code Of Conduct

`LizzieYzy Next` 希望成为一个让人愿意长期参与的项目。

我们欢迎直接、具体、有效的反馈，也希望讨论始终保持基本的尊重。

## Our Standard

在 issue、discussion、pull request、commit message 和其他协作场景中，请尽量做到：

- 聚焦问题本身，不攻击个人
- 给出复现步骤、上下文或取舍理由
- 尊重不同水平、不同语言背景、不同平台环境的使用者
- 接受别人指出问题，也愿意解释自己的判断
- 讨论分歧时尽量回到事实、日志、测试结果和用户影响

## Unacceptable Behavior

下面这些行为不适合出现在本项目中：

- 人身攻击、羞辱、嘲讽、辱骂
- 歧视性言论、骚扰、威胁、恶意挑衅
- 故意刷屏、带节奏、重复制造无效冲突
- 在没有必要的情况下公开他人隐私、账号信息或敏感数据
- 明知没有复现信息仍反复发布情绪化指责

## Maintainer Responsibilities

维护者会尽力：

- 让反馈入口保持清晰可用
- 在争议出现时推动讨论回到事实和方案
- 对明显破坏协作氛围的内容进行提醒、隐藏或关闭
- 优先保护愿意认真反馈和认真维护的人

## Reporting

如果你遇到明显不符合上述约定的行为，可以：

- 在相关线程中温和指出问题
- 通过 GitHub issue 或 discussion 联系维护者
- 如果涉及安全或隐私风险，请优先参考 [SECURITY.md](SECURITY.md)

## Scope

本行为准则适用于本仓库的公开协作空间，也适用于以本项目名义进行的沟通。

我们不追求形式化客套，但会认真维护一个友好、务实、能解决问题的协作环境。
