# Support

如果你在使用 `LizzieYzy Next` 时遇到问题，先选对入口，通常会比直接发一个模糊 issue 更快得到结果。

If you need help with `LizzieYzy Next`, start with the closest path below. Clear routing helps us answer faster.

## Where To Ask

| 你想做什么 | 推荐入口 |
| --- | --- |
| GitHub 下载慢，想直接用国内下载 | README 顶部的百度网盘公开下载入口 |
| 问怎么安装、怎么选包、怎么使用 | [GitHub Discussions](https://github.com/wimi321/lizzieyzy-next/discussions) |
| 想直接和中文用户讨论 | QQ 群 `299419120` |
| 反馈可稳定复现的软件问题 | [Bug Report](https://github.com/wimi321/lizzieyzy-next/issues/new?template=bug_report.yml) |
| 反馈安装成功 / 安装失败 / 首次启动异常 | [Installation Report](https://github.com/wimi321/lizzieyzy-next/issues/new?template=installation_report.yml) |
| 查看下载包 | [Releases](https://github.com/wimi321/lizzieyzy-next/releases) |
| 处理安全问题 | [SECURITY.md](SECURITY.md) |

## Before You Open A Report

请尽量准备这些信息：

- 操作系统和版本
- 你下载的安装包文件名
- 机器架构：Windows x64 / Windows x86 / macOS Apple Silicon / macOS Intel / Linux x64
- 是否使用普通 `with-katago`、`nvidia`，还是 `without.engine`
- 如果问题和野狐抓谱相关，请提供你输入的 **野狐昵称**
- 错误截图、日志、系统安全提示

## Fastest Path For Common Cases

- “我不知道该下载哪个包” -> 先看 [docs/PACKAGES.md](docs/PACKAGES.md)
- “GitHub 下载太慢” -> 先回到 [README.md](README.md) 顶部，直接使用公开百度网盘下载入口
- “我装不上 / 启动不了” -> 先看 [docs/INSTALL.md](docs/INSTALL.md) 和 [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)
- “我想知道某个平台有没有人实测过” -> 看 [docs/TESTED_PLATFORMS.md](docs/TESTED_PLATFORMS.md)
- “野狐抓谱没反应” -> 先确认你输入的 **野狐昵称** 是否正确，以及该账号最近是否有公开棋谱
- “我想知道项目接下来重点维护什么” -> 看 [ROADMAP.md](ROADMAP.md)

## Scope

这个仓库主要维护：

- 野狐棋谱同步
- 发布包与安装体验
- KataGo 集成
- 原项目常用分析流程

不保证支持每一种自定义引擎组合、每一种历史系统环境，或者所有第三方修改版配置。

## English Notes
